# file-organizer
organizes files based on filenames

organizes the current folder safely
```bash
python file_organizer.py
```

deep-scans and flattens all subdirectories
```bash
python file_organizer.py --recursive
```

previews changes without doing them
```bash
python file_organizer.py --dry-run
```

reverts everything
```bash
python file_organizer.py --undo
```
